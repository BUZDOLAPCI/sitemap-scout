# Dedalus Marketplace Info

## Short Description
Discover sitemaps and build crawl frontiers for any website

## Long Description
Find sitemaps via robots.txt and common paths, enumerate all URLs from XML sitemaps, and build filtered crawl frontiers with include/exclude rules. Tools: discover_sitemaps, list_sitemap_urls, build_crawl_frontier.

- RAG
